

# create venv

```shell
python3 -m venv venv

```
or
```shell
python -m venv venv
```

# activate venv

```shell
source venv/bin/activate
```

# install requirements
```shell
pip install -r requirements.txt
```


to run the server user

```shell
uvicorn main:app --host 0.0.0.0 --port 5000 --reload
```
